# Canonical Examples — Three Fully-Worked Prompts

Three complete prompts that passed iteration testing, stacked in the full 6-layer structure. Use these as templates — copy the structure, swap the object specifics, keep the closing block near-verbatim.

Each example is annotated so you can see which clause does which job.

---

## Example 1 — Alarm Clock (jelly shell, electronic-native)

> A single compact Y2K-styled bedside alarm clock floating in mid-air on a pure white #ffffff background, flat, no gradient, no contact shadow, no ground shadow, no surface, no spatial grounding. Three-quarter front angle, centered.
>
> The creative twist is a **semi-translucent tinted polycarbonate jelly shell** in saturated candy-mint — iMac G3 saturation, rich and candy-bright, fully saturated, NOT pale, NOT milky, NOT washed out. The interior chassis and speaker coil sit softly visible as faint darker shapes behind the tint — quiet interior geography, not crisp component detail.
>
> Per-component materials: the primary body shell is candy-mint translucent polycarbonate with a subtle satin sheen; the twin bells on top are brushed chrome steel with a crisp specular highlight; the bell striker between them is polished chrome; the clock face bezel is cream-colored ABS; the clock face itself is matte off-white with crisp black serif numerals and sharp black hands; the base strip (where the body meets the feet) is graphite-dark matte rubber; the feet themselves are cream ABS; the snooze button on top is a chunky round soft-touch rubber button in candy-red with a cream silkscreened **tongue-out `:P`** graphic dead-center — this is the real snooze button. Every boundary between materials is a crisp hard line, no gradient, no bleed.
>
> Electronic touch: below the clock face, a small rectangular LCD window shows diegetic text "SNOOZE 07:42"; a gently-lit green LED dot sits to its left; silkscreened labels "ALARM / SNOOZE / SET" run beneath the LCD in tiny cream-colored type.
>
> Y2K diegetic detail: a small printed model code "AC-0427 / 24H" silkscreened on the lower-right cheek of the shell; a foil barcode sticker with serial "SN 0042-M" on the back underside (barely visible from this angle); a tiny holographic QC sticker on the base strip; **one printed silkscreen decal of a round two-eye pair applied to the surface on the upper-left cheek of the shell, simple 1990s sticker style with black outline and white glint, NOT sculpted**; a small warning decal "9V DC ONLY" near the battery compartment; no cables.
>
> Critical material and component accuracy: every named component displays its assigned material with a crisp hard boundary where materials meet — no bleed, no monochromatic flattening. Labels and stickers sit ON the surface as real decals, not painted patterns.
>
> **Surface finish realism:** color saturation sits IN the material, not painted over it. PBR shading is physically accurate: correct specular response for each material type. Default to matte, satin, soft-touch, or eggshell finishes on primary bodies — reserve gloss for small functional components (knobs, buttons, LCDs, lens glass). Factory-fresh clean — no dust, no fingerprints, no scratches, no wear, no patina, no aging of any kind — but tactile and consumer-grade, not CGI-candy, not mirror-chrome. Tints are saturated, candy-bright, iMac G3 saturation — NOT pale, NOT milky, NOT washed out.
>
> **Staging:** the object floats in mid-air on pure white. NO contact shadow, NO ground shadow, NO plinth, NO surface implied. Assume no ground exists. NO base, pedestal, stand, or foot. NO cables, wires, or cords exiting the silhouette — wires contained within the object are fine.
>
> Styled as if shot on Phase One XT 55mm at f/5.6. Lighting: crisp directional studio key from upper-left; cool/warm controlled fill from the right.
>
> Backdrop: pure white (#ffffff), completely flat, no gradient. Aspect ratio 1:1, centered. Mood: quietly cheerful 7AM. Only the text/labels/stickers specified above, no other text, no watermarks, no additional objects.

**Annotations:**
- **Layer 1** (opening) — one sentence naming the object + staging lock
- **Layer 2** (creative twist) — jelly shell clause with softening + saturation lock
- **Layer 3** (per-component palette) — 9 components, each with assigned material and hard boundaries
- **Layer 4** (electronic touch) — LCD with diegetic text + LED + silkscreen labels + face button serving as snooze
- **Layer 5** (Y2K detail, 8 elements) — model code, barcode, QC sticker, eye decal, warning decal + face button (Layer 4) + LCD (Layer 4) + LED (Layer 4)
- **Layer 6** (closing block) — material accuracy + surface finish + shadowless/baseless + camera + aspect ratio + anti-text

Face pair used: **tongue-out `:P`** mouth + **round two-eye pair**.

The interior CRT tube neck, yoke coil, and chassis sit softly visible as faint darker shapes behind the tint — quiet interior geography, not crisp component detail.
>
> Form note: this is a late-1990s computer monitor (tall, gently tapered from front bezel back to the CRT tube, roughly 17" class), NOT a 1960s TV set. The front bezel is a tall rectangle with a slight inward curve on the CRT glass.

Default to matte, satin, soft-touch, or eggshell finishes on primary bodies — reserve gloss for small functional components (knobs, buttons, LCDs, lens glass). Factory-fresh clean — no dust, no fingerprints, no scratches, no wear, no patina, no aging of any kind — but tactile and consumer-grade, not CGI-candy, not mirror-chrome. ## Example 3 — Rubik's Cube (mixed approved-palette treatment)

The interior pivot mechanism sits softly visible as faint darker shapes behind the ice-mint shell edges — quiet interior geography, not crisp component detail.
>
> Per-component materials: the cube body is ice-mint translucent polycarbonate with a subtle satin sheen; the 27 facelets are individually-applied glossy enamel tiles in the six approved-palette colors, scrambled so no face is monochrome; the facelet edges sit slightly inset into the shell with a crisp hard boundary; the pivot hardware at the cube's internal axis is brushed chrome, visible only softly through the tinted body. ## Using these templates

1. Find the example closest to your target object (electronic-native → clock, screen-based → monitor, multi-panel → cube).
2. Swap the object specifics: silhouette, components, LCD text, model code, face pair.
3. Keep the **closing technical block near-verbatim** — it's load-bearing and does not need to be rewritten per prompt.
4. Check the face pair against the rest of your set (if any) — no two objects in the same set share the same mouth × eye combination.
